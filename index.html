How To Play

The goal of the game is to collect as many sets as possible. A set consists of some subset of the showing cards, where each color appears an even number of times. For example, the following cards form a set:

Each card collected in a set counts towards your score.

Controls

Use a mouse, touch, or keyboard input to select cards and activate the following buttons:

Enter

(Enter key) Enters your selection.

Clear

(Escape key) Clears your selection.

Reveal

("R" key) Reveals a solution.

Revealing a solution disables scoring for those cards.
